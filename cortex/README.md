# Cortex Workspace

Canonical execution-layer workspace for Nostra Cortex.

## Owns
- Runtime execution crates and services.
- Desktop execution shell and runtime adapters.
- Execution-domain libraries (`cortex-*`).

## Does Not Own
- Platform authority models and governance contracts (`nostra/*`).
- Nostra platform libraries (`nostra-*`).
- Research/reference metadata governance (`research/*`).

## Canonical Paths
- `apps/cortex-desktop`
- `apps/cortex-web`
- `apps/cortex-worker`
- `apps/cortex-gateway`
- `apps/cortex-eudaemon`
- `apps/cortex-git-adapter`
- `libraries/cortex-domain`
- `libraries/cortex-runtime`
- `libraries/cortex-agents`
- `libraries/cortex-ic-adapter`
- `libraries/cortex-workbench-contracts`
