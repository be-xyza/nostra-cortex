# Reference Topics

This registry lists the currently recognized reference topics for the main checkout.

| Topic | Scope |
|---|---|
| `agent-systems` | Agent runtimes, orchestration, safety, and coordination patterns |
| `data-knowledge` | Knowledge graph systems, semantic stores, and ontology tooling |
| `editor-platforms` | Editor and workspace platforms relevant to execution tooling |
| `icp-core` | Internet Computer core references and protocol-adjacent material |
| `misc` | Temporary or uncategorized research bundles |
| `tableviews` | Grids, table UIs, and structured browsing surfaces |
| `temporal` | Durable workflow engines and temporal orchestration patterns |
| `ui-substrate` | UI protocol, rendering substrate, and host-shell references |
| `workflow-orchestration` | Workflow engines, execution control, and runtime scheduling |

## Topic Policy

1. Prefer an existing topic when `topic_fit >= 4`.
2. Add a new topic only when repeated intake pressure justifies it.
3. Keep topic names kebab-case and capability-oriented.
